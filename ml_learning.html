Supervised Learning

In Supervised Learning, the algorithm learns from "labeled" training data. Think of it like a student learning from an answer key.

  • Input: Data with known results (Labels).
  • Task: Predict results for new data.
  • Example: Predicting if an email is "Spam" or "Not Spam".

Unsupervised Learning

Unsupervised learning looks for hidden patterns in data without labels. It’s like sorting a pile of various coins without knowing what they are—you'd naturally group them by size and color.
